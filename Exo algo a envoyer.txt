II.



The function in_array holds two parameters: element and list_elements.
The variable exist is false.


Loop: Until list-elements's length is reached, continue.
	if element equals list_element, then variable exist turns true.



translated to JS:
function in_array(element,list_elements) {
	var exist = false
	for (i = 0; i < list_elements.length; i ++) {

		if (element == list_elements[i]) {
			exist = true
		}
		
		else {
			exist = false
		}
	return exist
}
